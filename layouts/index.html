Self-healing container orchestration. Services restart automatically, rolling updates with zero downtime.

Offline-First

One-click install for self-hosted apps. Nextcloud, Gitea, Home Assistant, and more from the built-in catalog.

Get Started in Minutes

1

Flash or install CubeOS

Use Raspberry Pi Imager for a fresh install, or run the curl installer on an existing Linux system.

2

Connect to CubeOS WiFi (or your network)

CubeOS creates its own WiFi access point. Connect from any device, or plug in Ethernet.

3

Open http://cubeos.cube

The built-in DNS resolves cubeos.cube to the dashboard. Complete the setup wizard.

4

Install your first app

Browse the app store, pick an app, and deploy it with one click. That's it.
